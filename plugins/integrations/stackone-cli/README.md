# stackone-cli

Build and deploy custom StackOne connectors using the CLI and Connector Engine.

## Install

```bash
/plugin marketplace add stackonehq/agent-plugins
/plugin install stackone-cli@stackone-agent-plugins
```

## Use

- "Build a custom connector"
- "Deploy my connector"
- "Use the StackOne AI builder"
- "Set up CI/CD for connectors"
