# stackone-agents

Build AI agents that call StackOne-linked accounts via TypeScript/Python SDK, MCP, or A2A.

## Install

```bash
/plugin marketplace add stackonehq/agent-plugins
/plugin install stackone-agents@stackone-agent-plugins
```

## Use

- "Add StackOne tools to my agent"
- "Set up MCP with StackOne"
- "Integrate StackOne with OpenAI"
- "Build a multi-tenant agent"
