# stackone-unified-connectors

Build connectors that transform provider data into standardized schemas with consistent field names, enum values, and pagination.

## Install

```bash
/plugin marketplace add stackonehq/agent-plugins
/plugin install stackone-unified-connectors@stackone-agent-plugins
```

## Use

- "Start unified build for [provider]"
- "Build a schema-based connector"
- "Map fields to schema"
- "Test unified connector"
